sign

  • sign(text: string): string
    • Defined in src/ConvexAccount.ts:145

    Sign a hash message. This is called by the convex API class to sign a hash returned from the prepare api. This signed message cryptographically proves that the account owner has access to the private key.

    The API calls this with a hex string, that is converted to bytes, and then sigend. The resultant signed data is sent back as a hex string.

    Parameters

    • text: string

      Text hex string to sign

    Returns string

    A hex string signed with a prefix of '0x'
